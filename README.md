# Travel Planner 
Интерактивный планировщик путешествий для создания и управления маршрутами.
## Функционал
###  Поиск достопримечательностей
- Поиск по городам (в проекте только москва)
- Фильтрация по категориям:
  - Исторические места
  - Культурные объекты
  - Архитектура
  - Музеи

### Управление маршрутом
- Добавление мест в маршрут
- Перетаскивание для изменения порядка
- Удаление из маршрута
- Автоматический расчет времени

### Сохранение данных
- Автосохранение маршрута в LocalStorage
- Восстановление при перезагрузке страницы
- Сохранение последнего поиска

## Технические особенности

### Выполненные требования:
1. **Events** - 4 обработчика событий:
   - `submit` - форма поиска
   - `click` - добавление/удаление мест
   - `dragstart/dragend` - перетаскивание
   - `change` - выбор категории

2. **Асинхронные операции** - Имитация API запроса с Promise

3. **LocalStorage** - Сохранение всего маршрута

4. **Классовый подход** - Четкая структура с методами

5. **Чистый код** - Осмысленные названия переменных и функций

6. **Адаптивный дизайн** - Работает на всех устройствах

## Используемые технологии
- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5 для стилей
- Font Awesome для иконок
- LocalStorage для сохранения данных
